# Análisis de Notas en Java

## Descripción
Este proyecto consiste en un programa desarrollado en Java que analiza
un array de notas numéricas. El programa busca la nota más alta,
comprueba si dicha nota se repite, calcula la media de todas las notas
y evalúa si la media es superior o inferior a 5.

## Cómo ejecutar el programa
Pones los numeros que deseas analizar en el array y ejecutas el programa
